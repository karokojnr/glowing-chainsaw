# Pokedex

REPL game in GO using the Pokemon API.

## Usage:

- Fetch dependencies
    ```
    go get -d ./...
    ```

- Running the project
    ```
    go build && ./pokedexcli
    ```
- Testing
    ```
    go test ./...    
    ```
